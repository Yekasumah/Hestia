## Requirements For Initial Setup
- Install [NodeJS](https://nodejs.org/en/)

## Setting Up

-  Clone

### Install Dependencies:
- `$ npm install`

### Run tests

- `$ npx hardhat test`

### Start Hardhat node
- `$ npx hardhat node`

### Run deployment script
    In a separate terminal execute:
- `$ npx hardhat run ./scripts/deploy.js --network localhost`

### Start frontend

- `$ npm run start`
